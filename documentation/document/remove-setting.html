How to remove theme settings?

  1. remove "theme-setting-box" div from bottom of the every html pages
  2. remove theme-setting.js script file from bottom of the every html pages
  3. remove _theme-setting.scss file from scss/themes/theme-setting.scss folder
  4. Remove @import "themes/theme-setting"; from style.scss file
